# AGENTS.md — c2pa.net

## Build & Test

```bash
# Prerequisites: .NET 10.0 SDK, Rust toolchain, ClangSharp dotnet tool
# First time setup:
git submodule update --init --recursive
dotnet restore && dotnet tool restore

# Build (compiles Rust native lib via Cargo, generates bindings, then builds .NET)
dotnet build

# Run all tests
dotnet test

# Run a single test by name
dotnet test --filter "FullyQualifiedName~ReaderTests.TestMethod"

# Run tests in a specific class
dotnet test --filter "FullyQualifiedName~BuilderTests"

# Package for NuGet
cd lib && dotnet pack
```

## Architecture

This is a .NET wrapper over the [c2pa-rs](https://github.com/contentauth/c2pa-rs) Rust library, exposed via a C FFI boundary. The build pipeline is:

1. **Cargo builds the native library** (`c2pa-rs/` submodule → `c2pa_c.dll`/`.so`/`.dylib`)
2. **ClangSharp generates P/Invoke bindings** from `c2pa.h` → `lib/Bindings/*.cs`
3. **Roslyn source generator** (`generator/`) reads JSON schemas exported by the Rust `export_schema` crate and emits strongly-typed C# model classes at compile time
4. **High-level API** (`lib/*.cs`) wraps unsafe native pointers in disposable managed classes

Key design patterns:
- `ContextBuilder` → immutable `Context` → passed to `Reader` or `Builder` (shared config, signer, HTTP resolver)
- `[JsonSchema("path", "RootName")]` attribute on a `partial class` triggers the source generator to emit nested model types
- Native memory is managed via `IDisposable` wrappers around raw `C2pa*` pointers
- `C2pa.CheckError()` reads the last native error string and throws `C2paException`
- `NativeLibraryResolver` uses a `[ModuleInitializer]` to resolve the correct platform-specific native binary at runtime

## Conventions

- **Namespace**: `ContentAuthenticity` for public API; `ContentAuthenticity.Bindings` for generated P/Invoke layer
- **Unsafe code**: All native interop uses `unsafe` blocks with pointer types; classes that own native handles implement `IDisposable`
- **File header**: `// Copyright (c) All Contributors. All Rights Reserved. Licensed under the MIT License (MIT). See License.md in the repository root for more information.`
- **Target framework**: .NET 10.0 only (`net10.0`)
- **Central package management**: Package versions are defined in `Directory.Packages.props`
- **Testing**: xUnit with Moq; test fixtures come from `c2pa-rs/sdk/tests/fixtures/`
- **Code style**: 4-space indent, LF line endings, file-scoped namespaces (see `.editorconfig`)
- **Do not hand-edit**: `lib/Bindings/*.cs` (generated by ClangSharp) or source-generated model types (emitted by `generator/`)
